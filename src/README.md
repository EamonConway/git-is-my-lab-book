# Introduction

These materials aim to support
early- and mid-career researchers (EMCRs) in the
[SPECTRUM](https://www.spectrum.edu.au/) and [SPARK](https://www.spark.edu.au/) networks to develop their computing skills, and to make effective use of available tools and infrastructure[^infra].

## How to contribute

If you have any comments, feedback, or suggestions, please see the [How to contribute](how-to-contribute.md) page.
